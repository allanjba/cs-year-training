## CS Year Training

This repo is a structured, year-long coding training program.

### How it works

- **365 days**, **2 lessons/day**
  - **Lesson A (Foundations)**: CS fundamentals (data structures, algorithms, memory, complexity).
  - **Lesson B (Applied)**: real-world scenario work that applies Lesson A.
- Lessons are generated **one day at a time**, only when you ask.

### Structure

- `plan/`: curriculum plan versions (never overwrite; add new versions)
- `days/`: daily lesson folders
