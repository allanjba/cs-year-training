## Day 009 — Lesson B (Applied) review notes

### Core checks

- `totalViews`:
  - Sums all `views` correctly, including zeros.
- `averageLikesPerArticle`:
  - Returns 0 for an empty list.
  - Uses total likes / count with the right types.
- `tagFrequencyAcrossArticles`:
  - Flattens tags across all articles and counts correctly.

### Discussion prompts

- If you had to choose **one** or **two** numbers to show a non-technical
  stakeholder from this data, what would they be?
- How did you reuse earlier patterns (frequency maps, sums, averages)?
- Did you consider any edge cases like missing fields or empty tag arrays?

### Real-world tie-in

- These kinds of summaries drive dashboards and weekly reports.
- Understanding how to quickly extract and aggregate key metrics from raw
  data is a core skill for backend, data, and product engineers.\*\*\*
