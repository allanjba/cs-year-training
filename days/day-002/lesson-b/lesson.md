## Day 002 — Lesson B (Applied)

### Scenario: Support inbox triage

You are helping a small SaaS product team that manages customer emails through
a shared **support inbox**. They want a simple script that can:

- Mark messages as **urgent** based on certain keywords.
- Count how many messages match some basic conditions.

Today you will:

- Apply your string + conditional skills to a real-feeling task.
- Practice turning fuzzy rules into clear code.

### Concepts in play

- String search with `.includes()` (case-insensitive).
- Combining conditions with `&&` and `||`.
- Representing simple business rules in code.

### What you’ll implement in the exercise

- A function that decides whether a subject line should be treated as “urgent”.
- A function that counts how many messages are urgent within a small list.
- Some manual checks with `console.log` so you can see your classification.
